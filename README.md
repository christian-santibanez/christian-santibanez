Soy Ingeniero en Informática de INACAP Chile, con foco en **DevSecOps** y **Application Security**. Mi diferencial es entender el ciclo completo: diseño, desarrollo, despliegue y validación de seguridad.

Formación específica en desarrollo seguro: OWASP Top 10, ciclo de vida seguro del software, infraestructura como código (OpenTofu) y fundamentos de seguridad en cloud (Linux Foundation, AWS). ## Proyectos

### 🔍 [Sentinel API Analyzer](https://github.com/christian-santibanez/sentinel-api-analyzer)
Auditor OWASP API Top 10 activo con mapeo automático a MITRE ATT&CK y NIST CSF.

- 7 checks activos sobre endpoints REST: BOLA, Broken Auth, Mass Assignment, CORS, JWT, y más
- Pipeline DevSecOps con Bandit SAST + Trivy en GitHub Actions
- Backend FastAPI con rate limiting, logging estructurado JSON para SIEM y Docker multi-stage no-root
- Frontend React + TypeScript con WebSockets en tiempo real y exportación de reportes PDF

`FastAPI` `React` `TypeScript` `Docker` `PostgreSQL` `GitHub Actions` `Bandit` `Trivy`

- Autenticación JWT con access + refresh tokens y rotación segura

- CORS, CSRF y variables de entorno separadas por ambiente

`Django` `React` `DRF` `JWT` `Webpay` `Netlify` `Render`
